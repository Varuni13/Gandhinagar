# Gandhinagar-mobiResq

Gandhinagar Municipal Corporation dashboard prototype built with React and Vite.

## Local development

```powershell
npm install
npm run dev
```

Open `http://127.0.0.1:5174/`.

## Production build

```powershell
npm run build
```

## GitHub Pages

This repository includes a GitHub Actions workflow that builds and deploys the dashboard to GitHub Pages on pushes to `main`.
